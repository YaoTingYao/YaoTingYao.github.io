Cropland Maps on Plane-Scope Imagery Using DenseNet and U-Net at National Scales in Zambia

Yao-Ting Yao
Advisor: Lyndon Estes, Sitian Xiong
Master Thesis

Cropland Mapping is a crucial foundation for evaluating food security and land use dynamics. However, in Africa, the only source of cropland data are remotely sensed land cover maps because agriculture changes rapidly and lack of census cropland data.

This paper proposes to use U-Net sematic segmentation method to create Cropland Mapping in Republic of Zambia at national scales using high spatial-resolution Planet-Scope satellite imagery (2.77m) from June 2021 to March 2022.

Create Google Earth Engine(GEE) base labeling platform and workflow protocol

We created 200 by 200 grids covering Zambia, created catalog and used R script randomly seeded 600 labels. Base on growth-season imagery, off-season imagery and NDVI imagery, we digitized fields as labels. Used GEE API to create automation tools, include createing 3 classes raster label images(field, non-field and field boundry), imagery downloader and imagery sampler.

Model Training, Validation and Pridiction

600 labels created from Zambia and 4977 created from other Africa contry split 80% for Training and 20% for Validation.

U-Net is one of the cutting-edge architectures of CNN for image segmentation. U-Net has U-shape construction path (encoder) and an expansion path (decoder) structure, in which the spatial scale is subsequently reduced after consecutive pooling operations and increased in a contracting path. This research use Pytorch and scikit-learn library to train model and predict results. Choosed 100 epoch and 16 for batch size.
